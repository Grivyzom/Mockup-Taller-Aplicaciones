Sobre el Proyecto

Sistema de gestión integral para Cosmofood, un local de comida rápida en Valdivia. Este mockup presenta las interfaces diseñadas para digitalizar los procesos de venta, inventario y delivery.

Problemática Resuelta:
  • Gestión manual de pedidos que genera errores
  • Falta de control de inventario en tiempo real
  • Ausencia de canal digital para clientes
  • Procesos desorganizados en cocina y delivery
Tecnologías
HTML5 CSS3 JavaScript Bootstrap Tailwind UML
